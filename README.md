*Project Description
SmartFarm is a web-based Smart Agriculture platform designed to assist farmers with real-time decision support. It integrates weather forecasting, crop recommendation logic, market price advisory, and AI-based plant disease detection into a single unified system.
The platform aims to:
Improve farming productivity
Reduce crop loss
Provide market intelligence
Enable data-driven agricultural decisions
